문제 : 6064번 :"카잉 달력" (난이도 실버1)
풀이시간 : 9분 15초
한줄평 : 모듈러 연산과 관련있는 문제라고 생각해 학습하던 도중 아이디어가 떠올랐었다. 이미 아이디어가 떠오른 상태고, 구현이 매우 쉬워서 빠른시간에 풀 수 있었다. 모듈러 연산과 같이 수학공부가 어느정도 필요하다고 생각했다.
 
---알고리즘----
[핵심 아이디어]
answer = Mi + x = Nj +y이다.
즉, Mi + x를 N으로 나눈 나머지는 y가 된다. 이 값을 찾으면 된다.
i값을 점점 증가시키면서 찾으면 될 듯 하다.
 
[알고리즘 흐름]
1. testcase를 입력받는다. testcase횟수만큼 다음 과정들을 반복한다.
2. m, n, x, y를 입력받는다.
 2.1. 만약 x==m 인 경우 x = 0, y==n인 경우 y = 0이다.
3. i=0 ~ 40000까지 반복한다.
 3.1. temp = m*i + x
 3.2. temp > m*n인 경우 -1를 출력하고 반복을 종료한다.
 3.3. 만약 temp % n == y인 경우 temp를 출력한다.
